Código inline

Use const x = 10; para código inline.

Variáveis como variavel ou funções funcao() ficam destacadas.

Blocos de código

function exemplo() {
  const mensagem = "Hello, World!";
  console.log(mensagem);
  return true;
}

def exemplo():

    print(mensagem)
    return True
